querySelector(), querySelectorAll() : HTML 노드(관리객체)로 리턴하는 메서드

  • getElementBy** : HTMLCollection으로 리턴 (요즘 많이 사용)
    name, id 속성으로 접근이 가능.
  • querySelector : NodeList 형태로 리턴
    [0] index 번호로만 접근 가능
  • querySelector, querySelectorAll : 보다 더 구체적인 엘리먼트를 선택하고 싶을 경우
    getElement** : 더 빠르고 더 많은 지원을 받을 수 있음 (ex:id)
